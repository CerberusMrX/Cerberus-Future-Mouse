# Cerberus Future Mouse

Cerberus Future Mouse is a production-grade hybrid control system that combines hand gesture tracking and eye gaze tracking using a standard webcam to create a next-generation hands-free and touchless mouse replacement.

Authored by Sudeepa Wanigarathna.

## Features
- **Primary Control (Hand Tracking)**: Point to move the cursor, pinch to left-click, two-fingers to scroll.
- **Precision Layer (Eye Tracking)**: Micro-adjustments and UI element snapping using eye gaze direction and dwell clicks.
- **Hybrid Fusion Engine**: Intelligently blends hand and eye tracking data for the ultimate mix of speed and precision.
- **Auto-Calibration**: Adapts to lighting, camera geometry, and user physical traits.
